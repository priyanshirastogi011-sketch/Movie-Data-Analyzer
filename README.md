# Movie Data Analyzer

## Overview

Movie Data Analyzer is a Python project that analyzes an IMDb movie dataset and generates useful insights and visualizations.

## Features

* Data cleaning
* Statistical analysis
* Top-rated movies identification
* Genre analysis
* Runtime analysis
* Data visualization using Matplotlib
* CSV report generation

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib

## Project Structure

Movie_Data_Analyzer/

├── movies.csv

├── output/

│ ├── movies_per_year.png

│ ├── top_genres.png

│ ├── ratings_histogram.png

│ ├── runtime_histogram.png

│ ├── top10_movies.png

│ └── Top10Movies.csv

└── movie_data_analyzer.ipynb

## How to Run

1. Install required libraries:

pip install pandas numpy matplotlib

2. Open the notebook:

movie_data_analyzer.ipynb

3. Run all cells.

## Outputs

* Movies Released Per Year graph
* Top Genres graph
* Ratings Distribution Histogram
* Runtime Histogram
* Top 10 Movies graph
* Top10Movies.csv report
